Searching the Virtual Libray


Our search engine searches all the constituent sites of the Virtual Library. For the quickest search, just use the default options, type your search term and go. You can also modify the search parameters:

Match

All
All the words in the search field must be present in the document
Any
Any of the words must be present (you can enter several synonyms here)
Boolean
Explicit Boolean seach using AND/OR/NOT to define the query

Format

Long
Displays the titles, starting text and statistics on the matching pages.
Short
Displays just the page titles

Sort by

Score
Higher scoring pages are displayed first. Score depends on how many of the search terms match where in the document.
Time
Most recently changed documents are listed first
Title
Alphabetical indexing by title
Reverse...
The sorting by any of these criteria can be reversed.
